## ソフト説明

### ソフト名

無音でカッター


### 何するやつ？

ADVとか作るときに、声優さんのボイスを無音で自動カットしていくソフト。

将来的には色々機能乗せていきたい。


## 開発

### 依存性

```
nimble install nigui
```


### コンパイル

**通常**

```
nim c -r main.nim
```


**リリースコンパイル**

```
nim c -d:release --opt:size --app:gui main.nim
```
